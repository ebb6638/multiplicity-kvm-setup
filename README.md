Multiplicity 4 是 Stardock 出品的**软件版虚拟 KVM 切换器**，专门针对 Windows 多机协作场景。一套键盘鼠标，通过局域网，最多控制 10 台 PC。鼠标滑出屏幕边缘，自动切换到下一台机器。不需要任何连接线，不需要硬件设备，装完软件就能用。

## 为什么选软件 KVM，而不是硬件 KVM 切换器？

这是很多人在搜 KVM 方案时会纠结的地方。

硬件 KVM 切换器有一个根本限制：切换时显示信号要断开重连，至少需要 1-2 秒黑屏。高质量的 4K 硬件 KVM 价格往往在几百甚至上千元。线材也是个问题，每台被控电脑都要接 HDMI 或 DP 线拉到切换器，桌面一下子乱成一团。

软件 KVM 的逻辑完全不同。每台电脑保留自己的显示器，通过网络传输控制信号，切换几乎是瞬间的。不用额外买显示器，也不用走线。

不过软件 KVM 也有限制：所有设备必须在同一局域网内，并且目前主流方案（包括 Multiplicity）都只支持 Windows，Mac 或 Linux 需要另找方案。

对于多台 Windows 电脑都在同一张桌子（或同一个房间）的用户，软件 KVM 的体验远超硬件切换器。

## Multiplicity 4 的三种工作模式，分别适合哪些人？

### 无缝连接模式（最常用）

把几台电脑的屏幕位置在软件里设置好，就像拼一张大屏幕地图。鼠标推到屏幕边缘，自动滑进旁边那台电脑的显示器。键盘也跟着走。

这个模式适合桌面上摆着两到三台电脑、每台都有自己屏幕的用户。程序员在不同系统环境切换测试、设计师在两台机器上分开跑不同软件，都是典型场景。

### 无缝显示模式（专业版独有）

把另一台电脑的屏幕串流到当前机器显示。相当于把那台电脑变成一个远程扩展屏幕。

适合有一台性能机（主力工作站）和一台闲置笔记本的用户。游戏本跑大型渲染，旁边的老笔记本直接当第二块屏幕显示监控面板，主机性能分毫不浪费。

### KVM 模式（专业版独有）

在一台屏幕上以标签页形式集中管理多台 PC，支持鼠标键盘完整控制被控端。类似远程桌面，但延迟极低，适合 IT 运维或有多台无头服务器的用户。

## Multiplicity 4 的核心功能，逐条拆解

**跨机文件拖拽传输**

鼠标拖着文件，从 A 电脑的桌面拖过去扔到 B 电脑的文件夹里。就这么简单，不用网络共享，不用 U 盘，连微信传文件给自己都省了。据用户反馈，几个 GB 的文件通过有线局域网传输速度相当快。

**全局剪贴板同步**

在 A 电脑复制了一段文字或截了张图，移到 B 电脑直接粘贴，文字、图片、文件都支持。不用再截图然后用微信发给自己。

**音频重定向**

把被控设备的音频收集到主控机播放。对于在多台电脑上分别运行不同软件的用户来说，这个功能省去了戴两副耳机或频繁插拔音频接口的麻烦。

**AES-256 加密传输**

所有连接数据都经过 AES-256 加密，并且要通过密码验证才能建立连接。这一功能在 Multiplicity 4 中覆盖到了所有版本，不像上一代需要购买高级版才能用。

## 它和 Synergy、无界鼠标比，差距在哪里？

市面上的软件 KVM 方案主要有三种：

Synergy 是老牌选手，支持 Windows/Mac/Linux 跨平台。优点是跨平台，缺点是剪贴板只能共享文字，不能共享文件，拖拽文件传输也有问题，无线连接下偶尔有卡顿。

微软无界鼠标（Mouse Without Borders）免费，只支持 Windows，可以共享剪贴板并拖拽文件，但功能相对基础，稳定性不如商业软件，断线重连成功率不高。

Multiplicity 4 的优势在于：稳定性经过商业场景验证，文件拖拽传输流畅，剪贴板共享支持文字/图片/文件三种格式，加密标配所有版本，还有独特的无缝显示（串流副屏）功能。代价是只支持 Windows，并且需要付费购买。

如果你的设备全是 Windows，Multiplicity 的体验会比免费方案好一到两个档次。
